# Mecoria Media Script Standard v1.0

## Purpose

This document defines the official storytelling standard for all long-form YouTube videos produced by Mecoria Media.

Every Script Agent must follow this standard.

---

# Target Video Length

8–12 minutes

---

# Structure

1. Title

2. Thumbnail Hook

3. Opening Hook (0–20 seconds)

4. Introduction

5. Chapter 1

6. Curiosity Loop

7. Chapter 2

8. Major Twist

9. Scientific Explanation

10. Final Discovery

11. Ending

12. Call To Action

13. Visual Suggestions

---

# Storytelling Principles

- Start with curiosity.
- Delay important answers.
- Reward viewer attention.
- Build tension gradually.
- Keep every section connected.
- End with a memorable insight.

---

# Voice

- Professional
- Documentary
- Curious
- Intelligent
- Easy to understand

---

# Never

- Clickbait without payoff
- Fake facts
- Misleading titles
- Sensationalism
- Long unnecessary introductions

---

# Goal

Maximize:

- Audience Retention
- Watch Time
- Viewer Satisfaction
- Rewatch Value
