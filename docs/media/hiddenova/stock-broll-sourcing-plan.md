# Hiddenova Stock & B-roll Sourcing Plan v1

## Purpose

The first unlisted Hiddenova test proved that the production pipeline works, but the video felt too much like a slideshow.

The next public-quality version must be built around realistic video flow.

Target visual mix:

- 70% real stock / b-roll / staged footage
- 20% motion graphics / diagrams / map graphics
- 10% text overlays / title cards

Graphics should support the story. They should not become the main video.

---

## Priority Assets

### A001 - Hero Domestic Return Sequence

Asset type: b-roll  
Production method: manual capture or licensed stock  
Priority: high

Purpose:

Show the returned box as the recurring character of the story.

Needed shots:

- hands opening a plain cardboard box
- generic product being examined
- product going back into box
- tape being pulled
- box placed near a door
- blurred phone / return confirmation feeling
- doorstep handoff
- later second customer opening a similar plain box

Acceptance criteria:

- no brand logos
- no readable labels
- no barcodes
- no private information
- cinematic close-up feel
- shallow depth of field preferred
- neutral home environment
- realistic hand movement
- no exaggerated acting

Reject if:

- packaging has visible brand names
- labels or addresses are readable
- footage feels like a commercial
- scene looks staged in a cheap way
- hands/product look unnatural

Initial search queries:

- cinematic package return home hands packing cardboard box no logo
- customer returning online order cardboard box doorstep
- hands sealing cardboard box return package close up
- online shopping return package home b-roll

### A010 - Sorting / Conveyor / Scanner / Warehouse Automation

Asset type: stock video  
Production method: licensed stock  
Priority: high

Purpose:

Create the real logistics backbone of the video.

Needed shots:

- parcel sorting conveyors
- packages moving across belts
- warehouse scanners
- automated diverters
- rolling bins
- loading dock movement
- warehouse robots
- scanning tunnels
- time-lapse box accumulation
- night warehouse / logistics activity

Acceptance criteria:

- realistic logistics environment
- movement in frame
- no obvious company branding
- no readable labels / barcodes
- no exact facility signage
- can be reused across multiple sections
- visually matches dark documentary tone

Reject if:

- carrier logos are visible
- footage is too cheerful / corporate stock style
- camera movement is too generic
- product labels are readable
- footage looks staged or fake
- resolution is too low
- license is unclear

Initial search queries:

- automated parcel sorting conveyor scanners warehouse no logos
- logistics warehouse conveyor belt packages moving
- parcel sorting center scanner diverter b-roll
- warehouse automation packages conveyor stock footage
- package scanning tunnel warehouse robots

## Footage Quality Rules

Approved footage should feel:

- real
- cinematic
- documentary-like
- operational
- physical
- process-driven

Avoid footage that feels:

- like a corporate ad
- overly staged
- too bright and cheerful
- generic filler
- visibly branded
- legally risky
- unrelated to the narration

---

## Legal / License Rules

Every downloaded or purchased footage file must have a license record.

Each record must include:

- asset ID
- source platform
- footage title
- download URL or source URL
- license type
- download date
- allowed usage
- notes about restrictions
- local file path

Do not use any footage publicly unless license status is clear.

---

## Logo / Label / Barcode Rules

Reject or crop/blur footage containing:

- readable shipping labels
- barcodes
- QR codes
- tracking numbers
- personal addresses
- company logos
- branded uniforms
- facility signage
- private dashboards
- fake UI overlays

If the footage is excellent but contains small risky details, mark it as:

needs_blur_or_crop

---

## Production Strategy

First public-quality rebuild should not try to solve all 24 assets.

Start with these:

1. A010 - sorting / conveyor / warehouse automation
2. A001 - home return sequence
3. A012 - inspection table sequence

Then add supporting internal graphics:

- A002 map/network graphics
- A004 text overlays
- A007 decision diagrams

This gives the video a real visual backbone while keeping graphics as support.
