# Hiddenova Thumbnail Standard v1

Standard name:

`hiddenova_cinematic_v1`

## Core Formula

One dominant subject + cinematic tension +
huge short text + high contrast +
clean composition.

## Text Rules

- Use 2 to 4 words.
- Use ALL CAPS.
- Use very large bold typography.
- Use white and yellow.
- Use a strong black outline and shadow.
- Text should occupy approximately 35 to 55
  percent of the thumbnail.
- Mobile readability has the highest priority.

## Composition Rules

- Use one dominant main subject.
- Keep visual clutter low.
- Reserve strong negative space for text.
- The subject and text must not compete.
- The subject must connect directly to the video.
- Use dramatic cinematic lighting.
- Create mystery, tension, danger, or curiosity.

## Forbidden Elements

- Small text
- Long sentences
- Paragraph text
- Multiple competing subjects
- Logos
- Watermarks
- Embedded UI clutter
- Low contrast
- Generic stock layouts
- Unrelated objects

## Quality Priority

1. Readability
2. Curiosity
3. Strong subject
4. Clean composition
5. Brand consistency
6. Aesthetic polish

## Good Text Examples

- WHEN CABLES SNAP
- WHERE BAGS GO
- WHO MOVES IT
- HOW IT SURVIVES
- WHAT REALLY HAPPENS
- THE HIDDEN ROUTE

## Operational Rule

A beautiful thumbnail that is not readable fails.

A readable thumbnail that is generic fails.

A cinematic, readable, curiosity-driven thumbnail
with one dominant subject passes.
