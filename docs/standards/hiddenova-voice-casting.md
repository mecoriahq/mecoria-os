# Hiddenova Voice Casting Standard v1

## Purpose

The current Cedar voice was good enough for pipeline testing, but it is not locked as the permanent Hiddenova voice.

Hiddenova needs a repeatable voice selection process before public publishing.

## Rule

Do not standardize a voice across the channel until it passes founder review.

## Evaluation Criteria

Each voice sample should be scored on:

- premium documentary feeling
- clarity
- flow
- authority
- curiosity
- listener comfort
- low boredom risk
- Hiddenova brand fit

## Current Decision

Cedar is the baseline test voice, not the final channel voice.

Before re-rendering the public candidate video, run voice casting and select the strongest voice profile.
