# Mecoria Naming Standard

## Purpose

Define a consistent naming convention across Mecoria OS.

---

## Files

Use:

- lowercase
- hyphen-separated

Examples:

- script-standard.md
- media-engine.md
- agent-contract.md

---

## Folders

Use:

- lowercase
- no spaces
- singular names when appropriate

Examples:

- agents
- core
- configs
- workflows

---

## Python Files

Use:

snake_case

Examples:

- openai_client.py
- loader.py
- output_parser.py

---

## Python Variables

Use:

snake_case

Example:

channel_name
video_title
script_output

---

## Python Classes

Use:

PascalCase

Example:

ResearchAgent
ScriptAgent
OpenAIClient

---

## Constants

Use:

UPPER_SNAKE_CASE

Example:

OPENAI_API_KEY
DEFAULT_MODEL
MAX_RETRIES
