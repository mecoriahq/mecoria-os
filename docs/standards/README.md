# Mecoria Standards

Version: 1.0

---

## Purpose

This directory contains the official operating standards for Mecoria OS.

These standards define how AI agents create, review, optimize, and publish content.

Standards are provider-independent and apply across all Mecoria business units.

---

## Principles

- Standards are the single source of truth.
- AI agents follow standards.
- Standards evolve through production feedback.
- A rule is added to a standard before it is added to an agent.

---

## Current Standards

- Naming
- Image Generation
- Thumbnail
- Quality

---

## Future Standards

- Scripting
- SEO
- Publishing
- YouTube
- AI Agents
- Coding

---

## Development Rule

A quality rule must first be documented here before being implemented inside an AI agent.
