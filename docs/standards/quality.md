# Mecoria Quality Standard

Version: 1.0

---

## Purpose

Define the minimum quality requirements before any content moves to production.

---

## Requirements

Every asset must be:

- Accurate
- Complete
- Consistent
- Readable
- Production-ready

---

## AI Agent Rules

Every production agent must:

- Validate input
- Validate output
- Follow schema
- Respect company standards

---

## Continuous Improvement

Every production issue should result in an update to the relevant company standard.
