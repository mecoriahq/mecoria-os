# Script Agent

## Purpose

The Script Agent is responsible for transforming validated research ideas into production-ready long-form YouTube scripts.

It receives structured research output generated by the Research Agent and produces a complete script following Mecoria Media standards.

---

## Input

Research Agent JSON output:

```
agents/research/output/<channel>/latest.json
```

---

## Output

Latest script:

```
agents/script/output/<channel>/latest.json
```

Archived scripts:

```
agents/script/output/<channel>/archive/
```

---

## Responsibilities

- Read validated research output.
- Select the target idea.
- Generate a long-form YouTube script.
- Return structured JSON.
- Save latest output.
- Archive every execution.
- Never modify Research Agent output.

---

## Dependencies

- OpenAI API
- Research Agent Output
- Python
