# Strict Editorial QA Checklist

- [ ] Total narration is within the configured 8-12 minute word target.
- [ ] Hook creates immediate tension, paradox, consequence, or question.
- [ ] Strongest counterintuitive fact appears in the first 120 words.
- [ ] Introduction advances the story and does not restate the hook.
- [ ] Main sections form one cause-and-effect journey.
- [ ] Script uses concrete mechanisms and consequences.
- [ ] Script avoids unsupported precision and invented facts.
- [ ] Repeated abstract phrases and AI-style filler are low risk.
- [ ] Title is direct, accurate, and curiosity-driven.
- [ ] Thumbnail text is 2-4 ALL-CAPS words.
- [ ] Title and thumbnail add different information.
- [ ] SEO chapters array is empty before actual audio timing exists.
- [ ] Overall and every critical editorial threshold pass.
