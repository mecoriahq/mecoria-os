QA Agent Workflow

1. Read the latest Script Agent output.
2. Read the latest SEO Agent output.
3. Evaluate the script quality.
4. Evaluate SEO alignment with the script.
5. Evaluate title, description, tags, keywords, hashtags, and thumbnail text.
6. Identify issues.
7. Provide structured recommendations.
8. Decide whether the next agent should run.
9. Return only valid JSON following the schema.
