Image QA Agent Workflow

1. Read the latest Image Generation Agent output.
2. Locate the generated image file.
3. Validate that the file exists and is readable.
4. Validate the image format and size.
5. Read Mecoria Image Generation Standard.
6. Evaluate visual quality against the standard.
7. Identify visual issues.
8. Provide structured recommendations.
9. Approve only if the image is suitable for publishing.
10. Return only valid JSON following the schema.
