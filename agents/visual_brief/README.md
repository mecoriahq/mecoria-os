# Visual Brief Agent

## Purpose

Visual Brief Agent creates a reusable visual creative brief for Mecoria Media content.

It does not generate images or image prompts.

Its output can later be used by:
- Image Prompt Agent
- Thumbnail Generator
- Social Media Creative Agent
- Advertising Creative Agent
- Blog Hero Image Agent

## Input

agents/script/output/<channel>/latest.json

agents/seo/output/<channel>/latest.json

agents/qa/output/<channel>/latest.json

## Output

agents/visual_brief/output/<channel>/latest.json

agents/visual_brief/output/<channel>/archive/

## Responsibilities

- Define visual objective
- Define target emotion
- Define audience
- Define story moment
- Define headline candidates
- Define subject details
- Define background details
- Define composition
- Define lighting
- Define color direction
- Define visual constraints

## Not Responsible For

- Image generation
- Image prompt generation
- SEO metadata
- Script writing
- Publishing
