# Research Agent Checklist

## Output Quality

- [ ] Generates exactly 10 content ideas.
- [ ] Each idea has a clear title.
- [ ] Each idea has a one-sentence summary.
- [ ] Each idea has a defined target audience.
- [ ] Each idea explains why it has potential.
- [ ] Each idea includes difficulty: Easy, Medium, or Hard.

## Validation

- [ ] No duplicate ideas.
- [ ] No vague or generic topics.
- [ ] Output is in English.
- [ ] Output is clean Markdown.
- [ ] Output is ready for Notion import.

## Safety

- [ ] Does not invent facts.
- [ ] Does not include harmful or misleading claims.
