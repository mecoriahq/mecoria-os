# Research Agent Workflow

## Workflow

1. Receive channel profile.
2. Load content strategy.
3. Analyze previous content ideas.
4. Detect duplicate topics.
5. Generate new content ideas.
6. Validate output quality.
7. Return structured results.

## Success Criteria

- 10 unique content ideas.
- No duplicate topics.
- Clear English output.
- Structured Markdown format.
- Ready for Notion import.
